CineShare

Movie Review & Discovery Platform

โปรเจคพัฒนา Web Application สำหรับค้นหาและรีวิวภาพยนตร์

CineShare คืออะไร?

แพลตฟอร์มสำหรับผู้ที่ชื่นชอบภาพยนตร์เพื่อค้นหาข้อมูล รีวิว และแบ่งปันความคิดเห็น

  • ค้นหาข้อมูลหนังระดับโลกแบบ Real-time จาก TMDB Cloud API
  • ระบบรีวิวและให้คะแนน (1-5 ดาว) พร้อมแสดงค่าเฉลี่ยแบบ Interactive
  • ระบบจัดการ Watchlist และรายการโปรด (Favorites) ส่วนตัว
  • หน้า Profile และ Dashboard สรุปภาพรวมสำหรับผู้ใช้แต่ละคน

Technology Stack

Frontend

  • HTML5 / CSS3 / JS
  • Bootstrap 5
  • Chart.js (Dashboard)
  • Vanilla DOM Interactivity

Backend

  • Python 3
  • Django 5.2 Framework
  • Gunicorn WSGI Server
  • Requests API

Database

  • SQLite (Development)
  • Django ORM

System Architecture

ระบบถูกออกแบบด้วย 3-Tier Architecture พร้อม Cloud Integration:

  1. Presentation Tier: แสดงผลหน้าเว็บแบบ Responsive ด้วย Django Templates พร้อม UI แบบ Cinematic Dark Mode
  2. Application Tier: ประมวลผล Business Logic ผ่าน Django Views และสื่อสารกับ Services ภายนอก
  3. Data Tier: จัดเก็บข้อมูลผู้ใช้และรีวิวใน SQLite ในขณะที่ดึงข้อมูลภาพยนตร์มหาศาลผ่าน Cloud API แบบไม่ได้เก็บไว้ที่ระบบหลัก (Stateless)

Cloud & Service Integrations ☁️

  • TMDB Cloud API (SaaS):
    ดึงข้อมูลภาพยนตร์ยอดฮิต, ระบบค้นหา และฟิลเตอร์หมวดหมู่ได้อย่างครบถ้วนและแม่นยำ
  • TMDB Image CDN:
    ดึงรูปภาพโปสเตอร์และภาพคมชัดระดับสูงผ่าน CDN ทำให้ไม่ต้องเก็บรูปภาพใหญ่ๆ ไว้บน Server
  • YouTube Embed API:
    ฝังวิดีโอตัวอย่างภาพยนตร์ผ่าน Iframe (Trailer) ได้โดยตรง

Future Scale & Cloud Ready

การเตรียมพร้อมเพื่อยกระดับระบบขึ้น Cloud เต็มรูปแบบ

  • Managed Database: ย้ายจาก SQLite ไปใช้ Cloud PostgreSQL (เช่น AWS RDS หรือ Google Cloud SQL) สำหรับรองรับ User จำนวนมาก
  • Cloud Object Storage: จัดเก็บไฟล์เสียงและรูปภาพผู้ใช้ บนบริการอย่าง Amazon S3 อย่างปลอดภัย
  • Scalability (PaaS): ปรับใช้ Gunicorn เพื่อพร้อม Deploy แบบ Horizontal Scalability และบริการ CI/CD สำหรับ Auto Deploy
