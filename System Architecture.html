CineShare Architecture

ภาพรวมสถาปัตยกรรมระบบของโปรเจกต์ (N-Tier Architecture & API-Driven)

Layer 1: Presentation

Client Layer

HTML5 & CSS3

โครงสร้างและรูปแบบ

Bootstrap 5

Responsive UI Design

Vanilla JS / AJAX

Dynamic Interaction

Layer 2: Network

Edge & Delivery

Cloud Load Balancer

HTTPS / SSL Termination

WhiteNoise CDN

Static Files Server (CSS/JS)

Layer 3: Application

Django Backend

Gunicorn (WSGI HTTP Server)

Accounts App

  • User Authentication
  • Profile Management
  • Avatar Uploads

Movies App

  • Movie Browsing/Search
  • Watchlist & Likes
  • Reviews & Ratings

Services Layer (API Wrappers)

Django ORM

External

Cloud APIs

TMDB REST API

ดึงข้อมูล Metadata หนัง (ชื่อ, เรื่องย่อ, เรทติ้ง) แบบ Real-time

TMDB Image CDN

โหลดภาพโปสเตอร์และภาพพื้นหลังตรงจากเซิร์ฟเวอร์ TMDB

YouTube Embed

เล่นตัวอย่างหนัง (Trailers) ผ่าน iFrame ช่วยประหยัดแบนด์วิธ

Layer 4: Storage

Data Storage Layer

Relational Database

PostgreSQL (Prod) / SQLite (Dev)

Users, Reviews, Watchlists

Media Storage

Local Disk / Cloud S3

User Profile Avatars (/media/)

โครงสร้างนี้ออกแบบตามหลักการ Separation of Concerns ช่วยให้บำรุงรักษาและสเกลระบบได้ง่ายในอนาคต
